01 | Web Design

We collaborate in conceptualising web design or a product sales page, below is an idea of how we get started.

  1. Analysis – We talk about the project, it’s purpose, what is not working, how you wanted to improve and other Q&A related to the project.
  2. Wireframing – We focus on visual exploration to come up with best solution by arranging content elements using blocks based on your needs.
  3. Visual design – We create a UI styleguide in order to keep design consistency. From colors, fonts, spacing, padding and the usabillity of elements (button, forms, etc.)
  4. Development – After completing stage 1 to 3, I’ll start building a prototype of the finalized design turn it into an actual website and present for deployment approval.
  5. Discussion of the Final Project – Pros & Cons. Did it go as planned or should we start again.
