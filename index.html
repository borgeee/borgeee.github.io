02 | Web UI Design

 

My process when conceptualising Web UI design.

  1. Research - Collecting client information, the purpose of the project we are building, and other questions related to the project which will help during the process.
  2. Wireframing - On this process is where we collaborate ideas. We focus on visual exploration to come up with best solution arranging content elements using blocks based on user needs.
  3. UI/ Visual design - This is the part where we create a UI styleguide in order to keep consistency of designs and work closely with UX. From colors, fonts, spacing, padding and think about the usabillity of elements (button, forms, etc.)
  4. Repeat the process if neccessary.

You can also check my behance page for my design samples.
